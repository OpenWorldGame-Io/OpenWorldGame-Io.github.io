Open World


A community driven, open world "sandbox" project, for playing, testing and experimenting with Unreal Engine 5 and Lyra features and other Open Source projects and plugins.

Where



https://openworldgame.io

Play Now
Fork the Source Code
The Project and Planned Features


How


Currently, Open World is built using:
Unreal engine 5 + Lyra Starter Game + Open Source Community driven and reviewed Plugins.
UE5 gives a complete High End “Next Gen” game engine, with all Source Code that is publicly available. UE5’s Lyra Starter Game give a starting point for basic builds, characters, network play, etc.

When


Planned features will be added over time.
