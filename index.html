Open World


Can UE5's Lyra become a "Next Gen" Open Source, Open World, Community Driven "Metaverse" Environment?

Why


The "Metaverse" should be an open space where we can:
  • Build, Interact, Communicate, Experience, Play, Share

    • The world needs an open alternative to the Closed Source, Privately Owned Social
    Virtual Worlds and connected “Metaverses”

    How


    Currently, Open World is built using:
  • Unreal engine 5 + Lyra Starter Game + Open Source Community driven and reviewed Plugins

    UE5 gives a complete High End “Next Gen” game engine, with all Source Code that is publicly available

    UE5’s Lyra Starter Game give a starting point for basic builds, characters, network play, etc.

    • When


    Starting now! Features will be added over time...

    Where


    - Players can see and/or join the world from various devices, including:
    - PC Native clients (Windows, Mac)
    - Mobile Phone/Tablet
    - Browser, via Pixel Streaming
    - TV, Fridge, Toaster, Car, etc. e.g. via Video Streaming, with basic camera controls via mobile
    - etc.
